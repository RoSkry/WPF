XamlWriter - класс

Предоставляет одно статическое Save метод (перегрузки кратен), который может быть использован для ограниченной сериализации XAML, предоставленных объектов среды выполнения в разметки XAML.

Пространство имен:  System.Windows.Markup

Методы

 Save(Object)	Возвращает строку XAML, которая сериализует указанный объект и его свойства.
 Save(Object, Stream)	Сохраняет сведения о языке XAML в указанный поток для сериализации указанного объекта и его свойств.
 Save(Object, TextWriter)	Сохраняет сведения о языке XAML в качестве источника для предоставленного TextWriter объект. Выходные данные TextWriter может затем использоваться для сериализации предоставленного объекта и его свойств.
 Save(Object, XamlDesignerSerializationManager)	Сохраняет сведения о языке XAML в пользовательский сериализатор. Вывод сериализатора может затем использоваться для сериализации предоставленного объекта и его свойств.
 Save(Object, XmlWriter)	Сохраняет сведения о языке XAML в качестве источника для предоставленного XmlWriter объект. Выходные данные XmlWriter может затем использоваться для сериализации предоставленного объекта и его свойств.
