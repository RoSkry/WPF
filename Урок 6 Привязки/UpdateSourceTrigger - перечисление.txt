	Default	Значение по умолчанию UpdateSourceTrigger свойства цели привязки. Значением по умолчанию для большинства свойств зависимости является PropertyChanged, в то время как свойство Text имеет значение по умолчанию LostFocus.
	PropertyChanged	Немедленное обновление источника привязки каждый раз при изменении свойства цели привязки.
	LostFocus	Обновление источника привязки каждый раз при потере фокуса для элемента цели привязки.
	Explicit	Обновление источника привязки только при вызове метода UpdateSource.
