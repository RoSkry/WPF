		Методы

 GetBitmapEffect	Возвращает значение BitmapEffect для указанного объекта Visual. 
 GetBitmapEffectInput	Возвращает значение BitmapEffectInput для указанного объекта Visual. 
 GetChild	Возвращает дочерний визуальный объект с заданным индексом в коллекции заданного родительского объекта. 
 GetChildrenCount	Возвращает число дочерних объектов, содержащихся в заданном визуальном объекте. 
 GetClip	Возвращает отсеченную область для заданного Visual как значение Geometry. 
 GetContentBounds	Перегружен. Возвращает кэшированный ограничивающий прямоугольник для Visual. 
 GetDescendantBounds	Перегружен. Возвращает объединение всех ограничивающих прямоугольников содержимого для всех потомков визуального объекта, включая ограничивающий прямоугольник содержимого объекта Visual. 
 GetDrawing	Возвращает контекст рисования заданного Visual. 
 GetEdgeMode	Возвращает режим границ заданного объекта Visual как значение EdgeMode. 
 GetEffect	Возвращает эффект растрового изображения для указанного объекта Visual. 
 GetOffset	Возвращает смещение Visual. 
 GetOpacity	Возвращает непрозрачность Visual. 
 GetOpacityMask	Возвращает значение Brush, представляющее маску непрозрачности объекта Visual. 
 GetParent	Возвращает значение DependencyObject, представляющее родителя данного визуального объекта. 
 GetTransform	Возвращает значение Transform для Visual. 
 GetXSnappingGuidelines	Возвращает координату x коллекции направляющих (вертикальных). 
 GetYSnappingGuidelines	Возвращает координату y коллекции направляющих (горизонтальных). 
 HitTest	Перегружен. Возвращает самый верхний объект Visual проверки нажатия.
